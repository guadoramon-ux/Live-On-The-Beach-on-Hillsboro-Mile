
# Hillsboro Mile Beachfront Co-op Website

## Setup
1. Upload this project to your GitHub repository.
2. Enable GitHub Pages in your repo settings (source: `main` branch, `/root` folder).
3. Your site will be live at: `https://yourusername.github.io/repo-name/`

## Contact Form Setup (Formspree)
1. Go to https://formspree.io.
2. Sign up for a free account.
3. Create a new form and copy the **Form ID** (it looks like `xyzabcd`).
4. Replace `{your-form-id}` in `index.html` with your actual Formspree ID.
5. Test the form by sending yourself a message.

## Images
Optimized property images are stored in the `images/` folder.
